
# LoginAtomicDesign
Proyecto realizado siguiendo el patrón Atomic Design.

## Requisitos previos
- Node.js (v16 o superior recomendado)
- npm (v8 o superior)

## Instalación y uso
1. Clona el repositorio:
	```bash
	git clone https://github.com/SantiagoM0814/LoginAtomicDesign.git
	```
2. Entra a la carpeta del frontend:
	```bash
	cd LoginAtomicDesign/atomic-design-front
	```
3. Instala las dependencias:
	```bash
	npm install
	```
4. Inicia la aplicación en modo desarrollo:
	```bash
	npm run dev
	```
5. Abre tu navegador en la URL que aparece en la terminal (por defecto: http://localhost:5173)

## Notas
- La carpeta `node_modules` no se incluye en el repositorio, se genera automáticamente al instalar dependencias.
- Si tienes problemas, asegúrate de tener la versión recomendada de Node.js y npm.
